Welcome to Session #1: Rendering a list of items

Welcome! This is a take-homd coding challenge that recreates a small product catalog. It is designed to test your Angular skills, including:

  • Building components to match a UX Design
  • Ensuring responsiveness website
  • Standalone components (Angular 21)
  • TypeScript and typing
  • Angular routing
  • Async data handling with `HttpClient`

You will work with a small app that displays a list of items and their details. Your task is to extend the app with filtering and UI improvements.

Things to implement:

  • Update "items" view to load the items with the items.json file in /assets.
  • Implement filtering in the items listing
  • Click an item in the items listing will take you do the item-detail page.
  • Finish the item.service.ts file by implementing the stub methods.
